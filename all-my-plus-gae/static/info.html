Privacy Statement & Info

Information:

This script will read the public data available throught the Google+ API and present various statistics.

A nice summary of all the features has been created by +Johnathan Chung: j.mp/gplus-allmyplus


Privacy:

None of the retrieved information is saved on the site for longer than it takes to display the data to you.
Your authentication token is only saved temporarily to allow easy access to your data.


Contact:

If you have questions/suggestions/problems or want to stay updated on changes you can contact me on Google+.



Disclaimer:

This script is still in development and may be broken at times as I change stuff...


Known API issues:

  • Not all posts can be retrieved
  • The API sometimes times-out for users with lots of activity (sorry Robert and Tom)
  • For some people the API client throws an "invalid json in service response" error
  • Number of +1's and reshares isn't returned correctly all the time
